# PPT 视觉风格指南

本文档列出了项目支持的所有PPT视觉风格。每类风格提供独立的配色参考页面，包含完整的色值代码、应用示例和 accessibility 合规说明。

---

## 风格一览

| 风格名称 | 风格标识 | 适用场景 | 视觉特点 |
|---------|---------|---------|---------|
| [深色科技风](dark-tech.html) | `dark-tech` | 技术分享、开发者大会、产品发布 | 深色基底 + 蓝/紫/青渐变，科技感强烈 |
| [暖色系](warm-elegant.html) | `warm-elegant` | 商务汇报、教育培训、项目管理 | 暖白基底 + 珊瑚/琥珀色，亲和专业 |
| [薄荷清新风](mint-fresh.html) | `mint-fresh` | 环保/健康主题、创意工作坊、个人博客 | 浅色基底 + 翠绿/薄荷色，清新自然 |
| [霓虹赛博风](cyber-neon.html) | `cyber-neon` | 游戏开发、NFT/元宇宙、极客活动 | 深紫基底 + 霓虹粉/青，赛博朋克 |
| [极简纯净风](minimal-clean.html) | `minimal-clean` | 设计作品集、学术报告、企业官网 | 纯白基底 + 单色强调，极简专注 |

---

## 风格选择建议

```
┌─────────────────────────────────────────────────┐
│  受众与场景 → 推荐风格                            │
├─────────────────────────────────────────────────┤
│  CTO / 技术评审   →  深色科技风                   │
│  客户 / 管理层     →  暖色系 / 极简纯净风          │
│  创意 / 设计团队   →  薄荷清新风 / 霓虹赛博风      │
│  学术 / 教育       →  极简纯净风 / 暖色系          │
└─────────────────────────────────────────────────┘
```

## 调用方式

在 `ai-agent/ppt-generation-blueprint.json` 中，通过 `style_name` 字段指定：

```json
{
  "style_name": "warm-elegant"
}
```

AI Agent 会根据标识自动加载对应的配色方案（CSS 变量集）。
